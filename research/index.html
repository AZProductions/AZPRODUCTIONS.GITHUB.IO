Creating an Ipod manager.

Introducing… We still don’t have a name, but we have the groundworks layed out in a few simple steps.

  1. Start work on a library which supports reading and writing to all classic ipod types. (We are thinking to refactor LibGPod and make it .NET 6 compatible.)
  2. Start work on the main UI.
  3. Create a CLI and (G/C)UI cross-platform console application using multiple frameworks like Spectre.Console and Cocona.Lite.
  4. Release all programs to various platforms like the Microsoft Store and Ubuntu Package Manager.

Done! Thanks for reading this blog, if you have any suggestions, let us know on our website. (Photo by Brett Jordan on Unsplash)
